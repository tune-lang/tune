# T0201: unresolved name

A referenced name is not visible in the current scope or import set.

Define the name before using it, import the declaration that owns it, or use the fully visible receiver/member form.

Common causes:

- the value is private in another source file
- a module import was used, but the call forgot the namespace
- an enum variant was used without expected enum context

For enum variants, give Tune the enum shape at the use site:

```tn
enum Command {
  Move(Int, Int)
}

let command: Command = Move(3, 4)
```

For module imports, namespace imports do not leak members into top-level scope:

```tn
import "path"

let file = path.join("src", "main.tn")
```
