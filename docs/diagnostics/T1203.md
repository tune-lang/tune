# T1203: tag fact missing

A tag or compiler fact required by metadata analysis is missing.

Declare the tag, attach it to the intended item, or ensure the consuming meta query references an existing compiler fact.

Tags are ordinary declarations with typed fields:

```tn
tag route {
  path: String
}

@route(path = "/search")
let search(query: String): String = "query={query}"
```

If metadata code asks for a compiler fact such as `__params`, `__return`, or a
generated invoker, the declaration must be able to provide that fact. The error
should appear at the consuming meta use, not because the tag name itself is
special.
