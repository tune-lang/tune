# T0701: iteration len missing

A finite for source does not expose the required `len(): Size` contract.

Finite iteration needs a statically known length contract. Add or expose a compatible `len` member.

For v1, finite `for` is contract-based. A custom iterable needs both a length
member and indexed access:

```tn
struct Bag {
  items: [Int]

  len(): Size = self.items.len()
  Bag[index: Size]: Int = self.items[index]
}
```

This keeps `for item in bag` as ordinary typed member/index meaning instead of a
special loop-only path.
