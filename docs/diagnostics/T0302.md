# T0302: materialization failed

An unresolved literal or value could not commit to the required shape.

Numeric and sequence literals can stay flexible only while all constraints remain compatible.

Tune keeps literals flexible until context commits them:

```tn
let x = 20
let small: Byte = x
let normal: Int = x
```

But incompatible constraints still fail:

```tn
let x = 300
let small: Byte = x
```

When this happens, add the shape you actually want near the binding or change the
literal so it fits the target.
