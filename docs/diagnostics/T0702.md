# T0702: iteration index missing

A finite for source does not expose indexed access by `Size`.

Finite iteration needs an indexed access contract such as `value[index: Size]: T` or an equivalent sequence/range shape.

If `len(): Size` exists but indexed access is missing, Tune can count the items
but cannot prove how to get each item:

```tn
struct Bag {
  items: [Int]

  len(): Size = self.items.len()
  Bag[index: Size]: Int = self.items[index]
}
```

The loop variable gets its shape from the indexed access result.
