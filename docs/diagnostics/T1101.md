# T1101: public API inference

A public declaration exposes inferred facts that should be made explicit.

Add annotations to public parameters, returns, or exported shapes so the API remains stable and readable.

Private code can lean on inference:

```tn
let label(score) = if score > 30 => "pass" else "retry"
```

Public API should be stable for readers, tools, docs, and downstream modules:

```tn
pub let label(score: Int): String = if score > 30 => "pass" else "retry"
```

This warning does not mean Tune failed to infer the shape. It means the inferred
shape is becoming part of an exported contract.
